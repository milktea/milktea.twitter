milktea.twitter Installation
----------------------------

To install milktea.twitter using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``milktea.twitter`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        milktea.twitter

* Re-run buildout, e.g. with:

    $ ./bin/buildout
